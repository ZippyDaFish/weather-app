# Weather App
A simple and responsive weather application built with the OpenWeatherMap API.
This app allows users to search for current weather conditions in any city around the world.

### Technologies Used
- Frontend: HTML, CSS, JavaScript
- API: OpenWeatherMap API

### Features
- Global Search: Find weather information for any city worldwide.
- Real-Time Data: Displays current temperature, humidity, wind speed, and conditions.
- Dynamic Icons: Weather visuals that update based on current conditions (sunny, cloudy, rainy, etc.).
- Error Handling: Provides feedback for invalid locations or failed API requests.

### Example Usage
- Type in a city name (e.g., Paris) and instantly see:
- Temperature in Celsius/Fahrenheit
- Weather description (e.g., "Clear Sky")
- Icon representing the current conditions
- Additional details like humidity and wind speed
